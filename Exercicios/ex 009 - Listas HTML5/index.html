Listas

Listas ordenadas

A HTML chama de ordered lists todas aquelas listas onde a ordem dos itens é algo muito importante.

<li> Significa (list item) que identifica cada item da lista

  1. item 1
  2. item 2
  3. item 3
  4. item 4

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parâmetro são:

  • 1 - Valor padrão. Cria listas numeradas. Ex:1,2,3,4...
  • A - Cria listas alfabéticas em maiúsculas. Ex:A,B,C,D...
  • a - Cria listas alfabéticas em minúsculas. Ex:a,b,c,d...''
  • I - Cria listas com algarismos romanos em maiúsculas. Ex:I,II,III,IV...
  • i - Cria listas com algarismos romanos em minúsculas. Ex:i,ii,iii,iv...

Listas não ordenadas

Unordered lists são também conhecidas como listas demarcadas, são as listas onde a ordem dos itens não influenciará no significado da lista. Ela é apenas uma ótima maneira pra organizar os itens que não apresentam uma classificação necessariamente.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

O marcador padrão é a bolinha preta totalmente preenchida (circle), nas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

  • disc - padrão. Uma bola preta totalmente pintada
  • circle - Uma bola com uma borda preta e sem preenchimento
  • square - Uma pequeno quadrado preto totalmente pintado

Lista de Definições

É como se fosse um dicionário, temos termos e suas descrições.

Toda lista de definições está dentro de uma tag <dl> </dl> (definition list) Cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description). Assim como os itens da lista, essas duas últimas tags possuem fechamento opcional, seguindo a referência oficial da HTML5.

Vamos ver um exemplo simples que crua uma lista com três definições que já conhecemos bem aqui pelo curso.
